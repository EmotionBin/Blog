# 事件冒泡、事件捕获、事件委托

词穷就不多逼逼了，本文来了解一下 JavaScript 中的事件冒泡、事件捕获、事件委托机制  

## 事件冒泡

**事件冒泡**就是从当前触发的事件目标一级一级往上传递，依次触发，直到 document 为止  

单纯的看定义可能会有点抽象，举个例子，其实就像我敲击了一下键盘，我在敲击键盘的同时，我是不是也敲击了这台电脑
